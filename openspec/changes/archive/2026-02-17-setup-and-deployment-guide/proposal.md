## Summary

创建端到端的上手和部署文档，帮助新开发者快速搭建开发环境、申请云服务资源，以及将系统部署到生产环境。

## Motivation

当前项目缺少系统性的配置和部署文档。新开发者需要手动梳理代码才能了解如何运行系统，生产部署所需的配置修改也没有集中记录。

## User Value

- 新开发者可在 30 分钟内完成从零到本地运行
- 部署人员有明确的生产环境配置清单
- 配置变更可通过 OpenSpec 追踪，保持文档与代码同步

## Scope

- 云服务资源申请指南（DashScope API Key）
- 本地开发环境搭建（Server + Client）
- 生产环境部署教程（配置文件修改、Docker 部署）
- 配置参数速查表
